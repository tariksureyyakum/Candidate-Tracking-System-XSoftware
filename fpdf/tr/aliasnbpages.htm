AliasNbPages

AliasNbPages([string alias])

Tanımlama

Toplam sayfa sayısı için takma isim belirlenmesi için kullanılır. Döküman kapatılırkan toplam sayfa sayısı değeri otomatik olarak atanır.

Parametre

alias
Takma isim. Geçerli değer: {nb}.

Örnek

class PDF extends FPDF
{
function Footer()
{
    // alttan 1.5 cm geri git
    $this->SetY(-15);
    // Arial italic 8 fontunu seç
    $this->SetFont('Arial','I',8);
    // Geçerli ve toplam sayfa sayısını yaz
    $this->Cell(0,10,'Page '.$this->PageNo().'/{nb}',0,0,'C');
}
}

$pdf = new PDF();
$pdf->AliasNbPages();
