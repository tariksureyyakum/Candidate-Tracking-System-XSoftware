Header

Header()

Tanımlama

Bu method sayfa başlığının yapılandırılmasında kullanılır. AddPage() ve Close() ile birlikte çağırılır ve uygulama tarafından doğrudan çağarılmamalıdır. FPDF'teki geliştirme boştur. Bu yüzden kendinize özgü bir class geliştirmeniz gerekir.

Örnek

class PDF extends FPDF
{
function Header()
{
    // Select Arial bold 15
    $this->SetFont('Arial','B',15);
    // Move to the right
    $this->Cell(80);
    // Framed title
    $this->Cell(30,10,'Title',1,0,'C');
    // Line break
    $this->Ln(20);
}
}
