Image

Image(string file [, float x [, float y [, float w [, float h [, string type [, mixed link]]]]]])

Tanımlama

Sayfaya imaj eklemek için kullanılır. Boyutlu farklı şekillerde belirlenebilir:
  • genişlik ve yükseklik olarak (kullanıcı birimi şeklinde yada dpi cinsinden)
  • tek kenar uzunluğunu tanımlayarak. Bu durumda diğer kenar orantısal olarak hesaplanarak belirlenir.
  • belirli bir boyut tanımlanmazsa, imaj çözünürlüğünün 96 dpi olduğu durumlarda.
Desteklenen imaj formatları JPEG, PNG ve GIF dir. GD eklentisi GIF'e ihtiyaç duyar.

JPEG için tüm çeşitlere izin verilmektedir:
  • siyah beyaz
  • gerçek renk (24 bit)
  • CMYK (32 bit)
PNG için izin verilenler:
  • en çok 8 bitlik (256 renk) siyah beyaz
  • indexed color
  • gerçek renk (24 bit)
GIF için: Animasyonlu bir GIF kullanılıyorsa, sadece ilk görüntü kullanılır.

Transparanlık desteklenmektedir

Format belirtilebileceği gibi, dosya uzantısından da belirlenebilir.

Imaj üzerine köprü koymak ta mümkündür.

Hatırlatma: eğer imaj bir dosya içerisinde birden fazla kullanılıyorsa dosyaya sadece bir kez eklenir.

Parametre

file
Imaj dosyasının adı.
x
Sol üst köşenin x ekseni. Eğer tanımlanmazsa yada null girilirse, geçerli olan x koordinatı kullanılır.
y
Sol üst köşenin y ekseni. Eğer tanımlanmazsa yada null girilirse, geçerli olan y koordinatı kullanılır. Bunun yanında eğer sayfa bitirme fonksiyonu kullanılması gerekirse (otomatik sayfalama devrede ise de geçerli). Bu fonksiyon çağarıldığınca öncelikle sayfa ayırma sonra koordinat sayfanın en altı olarak atanır.
w
Imajın sayfa içerisindeki genişliği. Üç durumu vardır:
  • Eğer değer pozitif ise, kullanıcı birimindeki genişliği temsil eder.
  • Eğer değer negated ise, kesin değer dikey çözünürlüğü dpi olarak ifade eder.
  • Eğer belirtilmezse yada 0 girilirse otomatik olarak hesaplanır
h
Imajın sayfa içerisindeki yüksekliği. Üç durumu vardır:
  • Eğer değer pozitif ise, kullanıcı birimindeki yüksekliği temsil eder.
  • Eğer değer negative ise, kesin değer dikey çözünürlüğü dpi olarak ifade eder.
  • Eğer belirtilmezse yada 0 girilirse otomatik olarak hesaplanır
type
Imaj formatı. Olası değerler (büyük harf duyarsız): JPG, JPEG, PNG ve GIF. Eğer belirtilmezse, dosya uzantısına göre belirlenir.
link
URL veya AddLink() ten dönen tanımlayıcı.

Örnek

// 300 dpi çözünürlüğündeki logonun sol üst köşeye eklenmesi
$pdf->Image('logo.png',10,10,-300);
// URL den dinamik imaj eklenmesi
$pdf->Image('http://chart.googleapis.com/chart?cht=p3&chd=t:60,40&chs=250x100&chl=Hello|World',60,30,90,0,'PNG');
