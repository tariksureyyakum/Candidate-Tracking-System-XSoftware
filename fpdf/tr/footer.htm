Footer

Footer()

Tanımlama

Bu method sayfa altlığının eklenmesi için yazılmıştır. AddPage() ve Close() ile birlikte çağırılır ve uygulama tarafından doğrudan çağarılmamalıdır. FPDF'teki geliştirme boştur. Bu yüzden kendinize özgü bir class geliştirmeniz gerekir.

Örnek

class PDF extends FPDF
{
function Footer()
{
    // Go to 1.5 cm from bottom
    $this->SetY(-15);
    // Select Arial italic 8
    $this->SetFont('Arial','I',8);
    // Print centered page number
    $this->Cell(0,10,'Page '.$this->PageNo(),0,0,'C');
}
}
