Output

string Output([string name, string dest])

Tanımlama

Dökümanı verilen şu hedeflere göndermek için kullanılır: Yazı olarak, yerel dosya olark veya tarayıcıya. Son durum için varsa tarayıcıya özel bir eklenti kullanması gerekebilir. Yoksa dosyayı indirmeye zorlanır ("Farklı Kaydet" diyalog kutusu ile).
Method öncelikle Close()'u çağırır ve dökümanda sonlandırılması gereken işlemleri sonlandırır.

Parametre

name
Dosya adı. Eğer tanımlanmazsa, döküman doc.pdf adı ile tarayıcıya (hedef olarak I) gönderilir.
dest
Dökümanın gönderileceği hedef. Aşağıdaki değerlerden birini alır:
  • I: kullanmakta olduğu tarayıcıya gönder. Varsa eklenti kullanılır. Yoksa name ile tanımalanan isim kullanılarak "Farklı Kaydet" seçeneği ile PDF dosyası olarak link oluşturulur.
  • D: tarayıcıya gönderir ve dosyayı name değişkeni ile verilen isimde kaydetmeye zorlar.
  • F: name değişkeni ile verilen isimle yerel dosya olarak kaydeder.
  • S: dökümanı kelime dizisi olarak döndürür. name değişkeni gözardı edilir.

Bunlara da bakın

Close().
