AddFont

AddFont(string family [, string style [, string file]])

Tanımlama

TrueType, OpenType veya Type1 fontlarını ekleyip kullanılmasını sağlar. Öncelikle font tanımlama dosyasını makefont.php aracıyla tanımlamak gereklidir.
Tanımlama dosyası (ve gömülü hale getirilen font dosyası) font dizini içerisinde bulunmalıdır. Eğer bulunmazsa, "Could not include font definition file" (font tanımlama dosyası eklenemiyor) hatası oluşur.

Parametre

family
Font ailesi. İsim istenildiği gibi seçilebilir. Eğer standart bir aile ismi seçilirse, o isimdeki font üzerine yazılır.
style
Font stili. Olası değerler (büyük/küçük harf duyarlı):
  • salt kelime: regular
  • B: kalın
  • I: eğik
  • BI veya IB: kalın eğik
Geçerli değeri regular dir.
file
Font tanımlama dosyası.
Geçerli olarak, isim, küçük karakterlerle ve boşluksuz olarak aile ve stile göre yapılır.

Örnek

$pdf->AddFont('Comic','I');
ile aşağıdaki aynıdır: 
$pdf->AddFont('Comic','I','comici.php');

Bunlara da bakın

SetFont().
